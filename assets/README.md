# Assets Directory

This directory contains all images, icons, and other media assets for the Learning Oaks Library website.

## Structure

- `images/` - All image files (photos, illustrations, icons, etc.)
  - Profile images
  - Book covers
  - Illustrations
  - Icons

## Usage

When adding assets:
1. Place image files in the `images/` directory
2. Reference them in HTML/CSS using relative paths: `assets/images/filename.ext`
3. Use descriptive filenames (e.g., `jamie-oaks-profile.jpg`, `book-cover-1.png`)

## Supported Formats

- Images: `.jpg`, `.jpeg`, `.png`, `.svg`, `.gif`, `.webp`
- Other assets: `.pdf`, `.mp4`, `.mp3` (if needed)
